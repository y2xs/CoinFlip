# 🪙 CoinFlip
> A simple, addictive Python game to test your luck against the odds.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](./LICENSE)
![Python Version](https://img.shields.io/badge/Python-3.13-blue)
![Tech: Tkinter](https://img.shields.io/badge/UI-Tkinter-lightgrey)

**CoinFlip** is a minimalist desktop game built with Python. The challenge is simple: can you flip a coin and land on **Heads** ten times in a row? One "Tails," and your streak resets to zero!

---

## ✨ Features
* **🎯 Streak Tracking:** Real-time display of your current "Heads" streak.
* **📊 Statistics:** Tracks your total flips and your all-time high score.
* **🎨 Polished UI:** Built with Python’s Tkinter library for a clean, responsive experience.
* **🚀 Lightweight:** No heavy assets or external engines—just pure Python.
